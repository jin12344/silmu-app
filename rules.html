🐭 두더지 잡기 게임 규칙

온라인 2인 대전 · Firebase 실시간 동기화

기본 정보

👥

인원: 2명 (각자 다른 기기에서 접속)

🔄

라운드: 총 3라운드

⏱

시간: 라운드당 30초

🕳

구멍: 6개, 한 번에 두더지 1마리 등장

진행 방법

  • 1P 방 만들기 버튼을 누르면 4자리 방 코드가 생성됩니다.
  • 2P 방 참가하기 버튼을 누른 후 코드를 입력하면 게임에 참가합니다.
  • 두 명이 모두 입장하면 자동으로 카운트다운이 시작됩니다.
  • 두더지가 구멍에서 나타나면 최대한 빠르게 탭/클릭하세요!
  • 두 플레이어가 동시에 클릭하면 먼저 서버에 도달한 쪽이 점수를 얻습니다.
  • 두더지를 잡지 못하면 두더지가 숨고 새 두더지가 등장합니다.
  • 30초가 지나면 라운드가 종료되고 라운드 결과가 표시됩니다.
  • 3라운드가 모두 끝나면 총점이 높은 플레이어가 최종 우승합니다.

점수 계산

  • 두더지 1마리 클릭 = 1점
  • 놓친 두더지 = 0점 (감점 없음)
  • 3라운드 합산 점수가 높은 플레이어가 승리합니다.
  • 동점일 경우 무승부로 처리됩니다.

승리 조건

🏆

3라운드를 합산하여 더 많은 두더지를 잡은 플레이어가 우승합니다.
우승자의 닉네임과 축하 메시지가 화면에 표시됩니다.

플레이 팁

  • 두더지는 0.9~1.8초 동안만 나타납니다. 집중하세요!
  • 두더지가 어느 구멍에서 나올지는 완전 랜덤입니다.
  • 네트워크 지연이 낮을수록 먼저 점수를 차지하기 유리합니다.
  • 모바일에서 플레이 시 탭 반응 속도가 중요합니다. 화면을 준비하세요!
